# webRTC
Web Real-Time Communication



- It is a technology that enables real-time audio,video, and data sharing directly between browsers or devices without needing plugins or external software.


### NOTE :
- supports video calls , voice calls , screen sharing , and file transfer
- works using peer-to-peer connection , which reduces delay (low latency)
- requires a signaling server (usually node.js with webSocket/Socket.io) to establish the connection.
- uses STUN/TURN servers to connect users across different networks.
- Built into modern browsers like chrome , Firefox,  and edge


***WebRTC allows fast , secure , realtime communication directly between users in web or mobile applications***
